JavaScript (JS), and extensions of JS such as React and TypeScript, program the functional behavior of web pages. You can think of it like the muscles for a website.JavaScript is a programming language. HTML and CSS are markup languages, so they are technically not programming languages, but they are crucial for the look and feel of a website.This lab will focus on reviewing the basics of HTML and CSS, as well as a little JavaScript. By the end of the lab you will demonstrate these skills by creating a prototype web page of the ucsd.edu homepage. Everyone will create and submit their own page, but you are encouraged to pair up to discuss concepts and methods and learn from each other. Relevant online documentation is linked at the end of each task as a starting point for you to learn the required concepts. You can complete this lab at your own pace; for instance, if you are more experienced with web programming and are familiar with the concepts, feel free to skip or skim over the linked resources if you're able to accomplish each task on your own.
